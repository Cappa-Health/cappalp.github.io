Accessibility Statement

These pages are designed to meet WCAG 2.1 Level AA for keyboard access, semantic structure, and visible focus indicators.

Testing Checklist

  • Keyboard-only navigation across all links and controls
  • Visible focus indicator on every interactive element
  • Skip link moves focus to main content
  • Automated scan with Lighthouse or axe for each page
  • Manual screen reader smoke test on each page
